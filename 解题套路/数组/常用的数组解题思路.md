#### 数组中重复的数字

这题有两种做法，一种是使用HashSet去重，一种是使用双指针，我们来看下双指针做法。

```java
    public int findRepeatNumber(int[] nums) {
        Arrays.sort(nums);

        // 设置双指针
        int p1 = 0;
        int p2 = 1;
        for(int i = 0; i < nums.length-1; i++){
            if(nums[p1] == nums[p2]){
                return nums[i];
            }
            p1++;
            p2++;
        }
        return 0;
    }
```

首先将数组排序，接着设置两个指针，一前一后来遍历数组，如果相等的话，直接返回这个值就可以了。
