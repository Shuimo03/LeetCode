## 题目

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。输入一个递增排序的数组的一个旋转，输出旋转数组的最小元素。例如，数组 [3,4,5,1,2] 为 [1,2,3,4,5] 的一个旋转，该数组的最小值为1。  

示例 1：

输入：[3,4,5,1,2]
输出：1
示例 2：

输入：[2,2,2,0,1]
输出：0

## 解题思路

### 暴力解法

一开始并没有看懂题目，不是很懂它的旋转是什么意思，但是根据案例，大概弄懂了，只需要排序一次，然后输出数组第一个元素就可以了。

```java
    public int minArray(int[] numbers) {
        Arrays.sort(numbers);
        return numbers[0];
    }
}
```

但是这应该不是面试官想看到的。
